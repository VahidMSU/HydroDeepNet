Welcome to SWATGenX

Introduction

SWATGenX is an advanced platform for integrated watershed and groundwater modeling. It provides a comprehensive system to manage, simulate, and analyze water resources using state-of-the-art tools and technologies.

Core Components

SWATGenX integrates several key components to deliver robust and efficient water resource management solutions:

  • QSWAT+: Interface for SWAT+ model creation and management.
  • SWAT+ Editor: Tool for editing and configuring SWAT+ models.
  • SWAT+: Soil and Water Assessment Tool Plus, a comprehensive model for watershed simulation.
  • MODGenX: Groundwater model generation (MODFLOW-NWT) based on states and national datasets.
  • Flopy: Python package to work with MODFLOW models.
  • Parallel Processing System: Framework for calibration, validation, and sensitivity analysis using advanced algorithms like Particle Swarm Optimization and Morris Method.
  • GeoNet Deep Learning: Integration with deep learning models for enhanced data analysis and predictions.
  • Hierarchical Database System (HDF5): Efficient storage and management of large datasets.

Data Sources

SWATGenX utilizes a variety of data sources to ensure accurate and reliable simulations:

  • NWIS: National Water Information System
  • LOCA2: Localized Constructed Analogs Version 2 (2*2km resolution)
  • PRISM: Parameter-elevation Regressions on Independent Slopes Model (2*2km resolution)
  • NLCD: National Land Cover Database (30m resolution)
  • NHDPlus HR: National Hydrography Dataset Plus High Resolution (10m resolution)
  • NSRDB: National Solar Radiation Database (2*2km resolution)
  • SNODAS: Snow Assmilation Dataset (1*1km resolution)
  • gSSURGO: Gridded Soil Survey Geographic Database (30m resolution)
  • USGS 3D DEM: 3D Digital Elevation Model (10m resolution)
  • Water well information (Wellogic)

Groundwater Modeling

SWATGenX integrates groundwater modeling with the following components:

  • GW layers data
  • Hydraulic properties
  • Pumping rate
